Inheritance in CSS

In CSS, inheritance refers to what traits are recieved from other parts of CSS. Some general traits apply to multiple elements in an html code, while others only apply to specific elements. For example, if there is a paragraph element within a body of an html, applying a trait to the body will change everything in the body including the paragraph, while applying a trait to the paragraph will leave the rest of the body alone.

Specificity in CSS

Selectors are ways to group multiple different elememts and apply the same CSS code to it. Compound selectors group together different classes of elements, while multiple selectors combine all types of elements under the same category.

ID and Class Attributes

ID's and class selectors are ways of grouping up multiuple elements in CSS. ID selectors can be used to identify one element, while classes can be used to identify more than one element.
