Why we use the Normalize CSS Script

The normalize CSS files helps the HTML file look the same consistently across different browsers. There will always be a light variation on how html files look on each browser, so the normalize CSS reduces this difference as much as possible. This is done by stylizing many of the different elements to a default so they look consistent.

What is Semantic Markup

Semantic markup is a way of dividing the code of an html document to make it a readable and clean as possible. This includes dividing different elements of the code into different contatiners to make their inclusion in the code obvious.
